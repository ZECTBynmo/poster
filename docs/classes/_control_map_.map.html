map

  • map(keyactions: IDictionary<string | string[]>): void
    • Defined in control/map.ts:138

    Append event actions to the map.

    The map allows you to register actions for keys. Example: map.map({ 'ctrl-a': 'cursors.select_all', })

    Multiple actions can be registered for a single event. The actions are executed sequentially, until one action returns true in which case the execution haults. This allows actions to run conditionally. Example: // Implementing a dual mode editor, you may have two // functions to register for one key. i.e.: var do_a = function(e) { if (mode=='edit') { console.log('A'); return true; } } var do_b = function(e) { if (mode=='command') { console.log('B'); return true; } }

    // To register both for one key
Map.register('action_a', do_a);
Map.register('action_b', do_b);
map.map({
    'alt-v': ['action_a', 'action_b'],
});

    Parameters

    • keyactions: IDictionary<string | string[]>

    Returns void
